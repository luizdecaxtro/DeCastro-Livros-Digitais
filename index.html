📋 1. Aceitação dos Termos

Ao acessar e utilizar a Loja DeCastro, você concorda integralmente com estes Termos de Uso. Se não concordar com algum item, não utilize nossos serviços.

📱 2. Sobre os Produtos Digitais

  • Entrega Imediata: Produtos digitais são liberados automaticamente após confirmação do pagamento
  • Acesso Vitalicio: Uma vez adquirido, você terá acesso permanente ao produto
  • Formato Digital: Todos os produtos são entregues em formato digital (PDF, vídeo, software, etc.)
  • Compatibilidade: Verificamos compatibilidade com principais dispositivos e sistemas

💰 3. Pagamentos e Reembolsos

  • Métodos Aceitos: PIX (instantâneo) e Cartão de Crédito

  • Reembolso: Até 7 dias corridos para produtos não acessados/baixados
  • Problemas Técnicos: Garantia de 30 dias para questões técnicas

🚫 4. Uso Proibido

  • O uso comercial depende de licença específica (consulte-nos)
  • Violações podem resultar em suspensão e ações legais

🔒 5. Responsabilidades

  • Nossa: Entregar produtos conforme descrito e manter segurança dos dados
  • Sua: Usar produtos legalmente e manter credenciais seguras
  • Suporte: Oferecemos suporte técnico para instalação e uso básico
  • Atualizações: Produtos podem receber atualizações gratuitas

Política de Privacidade

🔒 1. Compromisso com sua Privacidade

A Loja DeCastro respeita sua privacidade e está comprometida em proteger seus dados pessoais conforme a Lei Geral de Proteção de Dados (LGPD).

📊 2. Dados que Coletamos

  • Dados de Cadastro: Nome, email, telefone para processamento de pedidos
  • Dados de Pagamento: Informações bancárias processadas por gateways seguros
  • Dados de Navegação: IP, browser, páginas visitadas para melhorar a experiência
  • Dados de Uso: Como você usa nossos produtos para oferecer suporte

🎯 3. Como Usamos seus Dados

  • Processamento: Para processar compras e entregar produtos
  • Comunicação: Envio de confirmações, suporte e atualizações
  • Melhorias: Análise para aprimorar produtos e serviços
  • Legal: Cumprimento de obrigações legais e fiscais

🚫 4. Não Compartilhamos

  • Seus dados NUNCA são vendidos para terceiros
  • Não enviamos spam ou emails não solicitados
  • Compartilhamento apenas com processadores de pagamento (obrigatório)
  • Dados anonimizados podem ser usados para estatísticas

🔐 5. Segurança

  • Criptografia SSL: Todos os dados em trânsito são criptografados
  • Servidores Seguros: Armazenamento em datacenters certificados
  • Acesso Restrito: Apenas funcionários autorizados acessam dados
  • Backups: Cópias de segurança regulares para prevenção de perda

🎯 6. Seus Direitos

  • Acesso: Solicitar cópia dos seus dados
  • Correção: Corrigir dados incorretos
  • Exclusão: Solicitar remoção dos dados (direito ao esquecimento)
  • Portabilidade: Receber dados em formato legível

💬 7. Cookies

Utilizamos cookies para melhorar sua experiência de navegação, lembrar preferências e analisar o tráfego. Você pode desabilitar cookies no seu navegador.
