⚙️ Configuração Mercado Pago

🚑 IMPORTANTE - Como obter as credenciais corretas:

  1. 1. Acesse mercadopago.com.br
  2. 2. Faça login na sua conta
  3. 3. Vá em "Seu negócio" → "Configurações" → "Credenciais"
  4. 4. Escolha "Produção" (para vendas reais) ou "Teste" (para testes)
  5. 5. Copie a Public Key e Access Token completos
  6. 6. Cole aqui sem espaços extras

⚠️ ATENÇÃO: Use credenciais de PRODUÇÃO para vendas reais e TESTE apenas para testes. Tokens de teste começam com "TEST-" e de produção com "APP_USR-".

⚠️ Se aparecer pontos (...........), cole o token completo novamente
